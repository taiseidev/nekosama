# テスト

<br>
テストを実施する際は併せてカバレッジを出力する

<br>

```
flutter test --coverage
```
<br>

**/coverage/lcov.info**ファイルが生成されます。次に**lcov.info**ファイルからカバレッジレポートのHTMLファイルを出力。
<br>

```
genhtml coverage/lcov.info -o coverage/html
```
